Growlのハンドラがあったら便利だと思い作成。

#deleteEmptyLine.applescript

指定されたテキストを改行ごとに配列に格納します。

#replaceText.applescript

#makeList.applescript

テキストを指定語句で区切り配列に格納する

#convertDate

#runningProcess.applescript
